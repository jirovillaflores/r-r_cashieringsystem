Instructions:

1. Open WAMP Server and type localhost/r&r_cashieringsystem.
2. If you do not have an existing account, sign up by creating email and password. Wait for confirmation.
3. If you are done creating an account, log in your credentials.
4. If you are done logging in as an admin, you may add your items, check inventory, and generate sales report on the sidebar.
5. If you are done logging in as a user, you may choose your items to be ordered and click Submit.
6. If you wish to update your item, click Update.
7. If there are items to be deleted, click Delete.
8. If you wish to proceed, wait for your receipt to be generated.
9. The admin will validate your order after payment.
